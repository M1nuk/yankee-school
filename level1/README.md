## Q. 네이버 실시간 검색어를 크롤링!

### 목표
* 10초마다 갱신이되어야 함.
* 글자색이 노란색이되어야 함.
* 검색어 순위와 텍스트가 일치해야함.
* 1~9위는 앞에 0을 자동으로 채워야 함.
* 10초 후 갱신되면 지난 내용은 지우고 새로써져야함.
* 순위와 내용 둘다 파싱을 해야함.

~※ Parsing 해오면 100% 확률로 중복값이 존재하는데, 이때 중복값을 제거하시오.~  
※ 네이버의 메인페이지 개편으로 중복값이 존재하지 않음

◎ 완성이되면 설명을 할 수 있도록 과정을 그림과 글로 서술하시오.
